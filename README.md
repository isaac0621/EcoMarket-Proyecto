# EcoMarket-Proyecto

Una plataforma web de mercado electrónico desarrollada con HTML, CSS y JavaScript, diseñada para ofrecer una experiencia de comercio digital funcional y organizada. Este proyecto incluye interfaces de navegación, manejo de productos y páginas clave como catálogo, ofertas y contacto, demostrando habilidades en desarrollo front-end y estructuración de aplicaciones web básicas. El código está disponible públicamente para evaluación y colaboración, y refleja enfoques de diseño responsivo y modular en un contexto de e-commerce.
